了解CloudDo

   项目介绍

  • 面向学习型的分布式微服务框架，精简配置，降低学习成本，减少过渡封装，展现技术本质
  • 基于spring cloud微服务框架，you build it,you run it
  • 服务的注册与发现（Eureka），服务消费者（Feign），路由网关(zuul)，服务鉴权（jwt）等
  • 一起开启微服务的学习之旅
